FAQ

1Can I get a loan on/through ByajBook?

Currently Byaj Book only deals in loan management, you can add your loan records and record payments but you cannot get a loan on byaj book

2 How many free loans can I create with Byaj Book?

You can add unlimited loans for free on Byaj Book, we will never charge you for adding loans and transactions.

3How interest is calculated?

By default the interest is calculated day wise. However, we will be adding an option to change the interest calculation frequency to match your interest cycle.

4Can I change loan details?

Only the Lender can change the loan details but the borrower can request for any changes in the loan to the Lender. To make a change or request a change you can tap on the particular loan and edit the loan from details page.

5 How multiple loans to the same person are managed/recorded?

As different loan can have a different rate of interest and time period all loans will be recorded separately even if it's given to the same Person. However you can check summary for a particular user by going to users profile from people section.

6 What if I don't want to see a loan added by someone else on my name?

If you don't want to see any loan added by someone else in your account then you have an option to hide it from the loan details page and if you want to restore it back you can go to your Profile-> Hidden Accounts Option and restore your loan
